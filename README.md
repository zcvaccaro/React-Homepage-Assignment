# React Portfolio Home Page

This class assignment is a responsive and interactive home page built with React, designed to be a part of a personal developer portfolio.

## Technologies Used

*   **React:** A JavaScript library for building user interfaces.
*   **Create React App:** Used to bootstrap the development environment.
*   **CSS:** For styling the components and layout.

## Setup and Installation

To get a local copy up and running, follow these simple steps.

1.  Clone the repository:
    ```sh
    git clone <your-repository-url>
    ```
2.  Navigate into the project directory:
    ```sh
    cd React_Home_Page
    ```
3.  Install NPM packages:
    ```sh
    npm install
    ```

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode.<br />
Open http://localhost:3000 to view it in your browser.

The page will reload when you make changes.

### `npm test`

Launches the test runner in interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder.<br />
It correctly bundles React in production mode and optimizes the build for the best performance.

Your app is ready to be deployed!
